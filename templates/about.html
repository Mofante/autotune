Audio Correction and Effects Application

Overview

This pitch correction application is an audio processing tool that leverages signal processing techniques to manipulate vocal audio. Built using Python, the application provides users with two pitch correction methods and creative audio effects.

Key Components

The application consists of three primary Python modules:

1. Flask Web Application (app.py)

The web interface allows users to:

  • Upload audio files (supporting WAV, MP3, OGG, and FLAC formats)
  • Choose pitch correction methods
  • Apply optional audio effects
  • Download processed audio files

Upload and Processing Workflow

  1. Generates a unique filename for each uploaded file
  2. Supports two pitch correction strategies:
    • Nearest note correction
    • Scale-based correction
  3. Provides optional audio effects with adjustable intensity

2. Autotune Core (autotune.py)

The heart of the pitch correction system, implementing audio processing techniques:

Pitch Correction Methods

  • Nearest Note Correction: Rounds pitch to the closest MIDI note
  • Scale-Based Correction: Maps pitch to the nearest note within a specified musical scale

Audio Effects

The application offers five creative audio effects:

  1. Robot Voice: Quantizes audio amplitude for a mechanical sound
  2. Chorus: Adds slightly pitch-shifted and delayed audio copies
  3. Distortion: Hard clips the audio signal for a gritty texture
  4. Tremolo: Applies amplitude modulation
  5. Reverb: Simulates sound reflection using multiple delay lines

3. Pitch Estimation (my_pyin.py)

A custom implementation of the Probabilistic YIN (PYIN) pitch estimation algorithm, which:

  • Detects fundamental frequencies in audio frames
  • Distinguishes between voiced and unvoiced segments
  • Uses signal processing techniques like cumulative mean normalized difference function and parabolic interpolation

Code Architecture

The application follows a modular design:

  1. Pitch detection (my_pyin.py)
  2. Pitch correction (autotune.py)
  3. Web interface (app.py)

Each module has a specific responsibility, making the code maintainable and extensible.

Conclusion

This pitch correction application demonstrates the power of digital signal processing in audio manipulation. By combining algorithmic techniques with an intuitive web interface, it provides the user with a versatile tool for creative audio transformation.
